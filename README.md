# ToDo List

Простое веб-приложение для управления списком задач, разработанное с использованием ASP.NET Core (Razor Pages) и Entity Framework Core.

## Технологии
- **ASP.NET Core 9.0** (Razor Pages)
- **Entity Framework Core 9.0**
- **SQLite** (база данных)
- **ASP.NET Core Identity** (аутентификация и авторизация)

## Требования для запуска
Для успешного запуска проекта вам потребуется:
- Установленный [.NET 9.0 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/9.0)

## Инструкция по запуску из исходников

1. **Откройте терминал (или командную строку)** и перейдите в папку с проектом `todoList` (там, где находится файл `todoList.csproj`).
   ```bash
   cd путь/к/папке/todoList/todoList
   ```

2. **Соберите проект** (эта команда автоматически восстановит все необходимые NuGet-пакеты):
   ```bash
   dotnet build
   ```

3. **Запустите приложение**:
   ```bash
   dotnet run
   ```

> **Примечание:** База данных SQLite (`app.db`) создается и обновляется автоматически при первом запуске приложения благодаря встроенному механизму автоматических миграций в `Program.cs`. Вам не нужно вручную запускать команды `dotnet ef database update`.

4. **Откройте приложение в браузере**. После запуска в консоли появятся ссылки (например, `http://localhost:5xxx` или `https://localhost:7xxx`). Перейдите по одной из них, чтобы начать пользоваться приложением.
